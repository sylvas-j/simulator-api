Court Case Simulator is built on a model trained with different coust cases result with datasets gotten from sherloc website.
The dataset consist of three features (text, crime types and sentence status).
To used the simulator:

  1. Just paste your court case text into the text box
  2. Or you can upload your cases from a csv or excel file using the upload button.
NB: Your text in an excel file should start from (A1 ceil) down; only on the A column without any formatting
